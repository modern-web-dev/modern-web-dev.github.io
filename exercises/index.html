Exercise HTML

  • Create an HTML document (index.html) having a header, a footer and a main content area. Make use of the <header> and <footer> elements.
  • The header contains a navigation link to the home page (make use of the <nav> and <a> elements); the footer contains copyright information.
  • Add a <form> to the main content area. The form consists of two <input> fields (first and last name) along with <label>s and a <button>

Block and inline elements (styles)

Add CSS styles so that the following is rendered: 


<!DOCTYPE html>
<html lang="en">
<head>
  <title>CSS Exercise 1</title>
  <style>/* Add styles here */</style>
</head>
<body>
  <!-- Do not modify HTML -->
  <div><div>1</div><div>2</div></div><div><div>3</div><div>4</div></div>
</body>
</html>
